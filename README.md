# 📊 Customer Behavior Analysis

📌 Project Overview

This project analyzes customer purchasing behavior using Python, SQL, and Power BI to extract meaningful insights and support data-driven business decisions.

It follows a complete end-to-end data analytics workflow — from raw data processing to dashboard creation and final reporting.

🎯 Objective

- Understand customer purchasing patterns
- Identify high-value customers
- Analyze sales trends and category performance
- Provide actionable insights to improve business strategy

🛠 Tools & Technologies

- Python (Pandas, NumPy) – Data cleaning & analysis
- Matplotlib / Seaborn – Data visualization
- MySQL / PostgreSQL – Data querying & analysis
- Power BI – Interactive dashboard

📂 Dataset

- Business-oriented structured dataset
- Includes categorical, numerical, and time-based features
- Cleaned for missing values, duplicates, and inconsistencies

🔍 Project Workflow

1. Data Loading

- Imported dataset into Python
- Checked structure, data types, and summary statistics

2. Exploratory Data Analysis (EDA)

- Identified trends and patterns
- Visualized distributions and relationships
- Detected outliers

3. Data Cleaning

- Handled missing values
- Removed duplicates
- Standardized formats

4. SQL Analysis

- Stored processed data in database
- Performed aggregations and filtering
- Used joins and group-by for insights

5. Dashboard Development

- Built interactive Power BI dashboard
- Included KPIs, filters, and charts
- Designed for business usability

📊 Key Insights

- High-value customers contribute the majority of revenue
- Certain product categories perform better than others
- Seasonal trends impact customer purchasing behavior
- Discounts influence buying frequency

▶️ How to Run

Prerequisites
- Python 3.x
- MySQL / PostgreSQL
- Power BI Desktop

Steps
1. Clone the repository
2. Install dependencies:
   pip install pandas numpy matplotlib seaborn
3. Run Python scripts
4. Load data into SQL
5. Execute SQL queries
6. Open Power BI dashboard
